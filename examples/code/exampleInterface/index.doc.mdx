# movePoint

`code/exampleInterface`
**AST Node Type:** *FunctionDeclaration*

> Function that moves a point by a specified delta.

#### Parameter:

- `point` *Point* — The point to move.
- `dx` *number* — The change in the x-coordinate.
- `dy` *number* — The change in the y-coordinate.
#### Returns:
`Point`  new position of the point.

#### Surse Code:

```ts
function movePoint(point: Point, dx: number, dy: number): Point {
    return { x: point.x + dx, y: point.y + dy };
}

```
