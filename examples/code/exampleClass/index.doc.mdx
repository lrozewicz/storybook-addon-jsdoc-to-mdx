# Rectangle

`code/exampleClass`
**AST Node Type:** *ClassDeclaration*

> Represents a rectangle.

#### Source Code:

```ts
class Rectangle {
    private width: number;
    private height: number;
    constructor(width: number, height: number) {
        this.width = width;
        this.height = height;
    }
    area(): number {
        return this.width * this.height;
    }
}

```
